Monet developers guide
======================

The Monet source code is maintained in https://gitlab.cern.ch/lhcb/Monet. 

Monet is a web application based on the `Flask framework <https://flask.palletsprojects.com/en/3.0.x/>`_
and on `bokeh <http://bokeh.org/>`_ for the display of the data.
